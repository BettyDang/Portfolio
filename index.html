About

Hey there, I'm Betty Dang, I am a Full-Stack Developer and Computer Science student at LaSalle College in Montreal with a passion for building modern, efficient web applications. My development journey began with HTML5 and front-end technologies, and has since evolved into full-stack development involving database design, backend systems, and dynamic user interfaces.

I work with technologies including JavaScript, Java, PHP, C#, SQL, HTML, CSS, Bootstrap, and modern development tools to create scalable and user-focused applications. Through academic projects such as ShopEasy, an e-commerce web application, I have gained hands-on experience designing relational databases, implementing backend functionality using Java and JDBC, and developing administrative interfaces for managing application data.

Before transitioning into software development, I worked in healthcare where I developed strong analytical thinking, precision, and problem-solving abilities—skills that now support my approach to building reliable and well-structured software solutions.
